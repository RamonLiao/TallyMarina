# 品牌名稱建議

核心概念：AI 輔助的數位資產子分類帳，將鏈上活動拆解為可解釋、可核准、可對帳及可匯入 ERP 的會計分錄，並把每期關帳快照錨定成不可竄改的審計證據鏈。

## 1. TallyMarina

`Tally` 代表核算、加總與帳務確認；`Marina` 是讓船隻安全停泊及管理的港灣。

名稱傳達「讓分散的鏈上資產活動進入可管理的企業財務港灣」，語氣親近但仍保有 B2B 軟體的專業感。適合強調資料接入、分類及月結工作台。

## 2. LedgerShoal

`Ledger` 直接表達分類帳定位；`Shoal` 既可指魚群，也可指淺灘。

魚群意象呼應大量鏈上交易被協調成一致的財務事件；淺灘意象則代表把原本深奧、難以理解的鏈上資料帶到財務團隊能清楚檢視的範圍。適合強調多來源資料標準化。

## 3. CloseHarbor

`Close` 對應 period close／月結；`Harbor` 代表安全靠港。

名稱直接抓住財務團隊最重要的工作成果：讓每個會計期間帶著完整對帳、核准及證據安全關帳。商業語意清楚，是五個名稱中最適合企業財務 SaaS 定位的選項。

## 4. DeltaVerity

`Delta` 是多條河流匯聚及分流形成的三角洲，象徵錢包、交易所、協議與 ERP 等異質資料來源；`Verity` 代表真實性與可驗證性。

名稱突出產品把多股資料流轉換成可信財務事實的能力，也能涵蓋 manifest hash、Merkle root 與 append-only AuditAnchor。適合強調審計與資料 lineage。

## 5. CurrentLedger

`Current` 同時具有水流與「當期」的雙重含義；`Ledger` 表明產品是 ERP 上游的數位資產子分類帳。

名稱可以解讀為「持續接收鏈上資金流的分類帳」，也可以連結 current period accounting。適合強調即時資料接入、期間管理及鏈上資產流向。

## 推薦排序

1. **CloseHarbor** — 商業定位最清楚，能直接連結關帳、可靠性與企業客群。
2. **DeltaVerity** — 技術與審計辨識度最高，適合強調可驗證證據鏈。
3. **TallyMarina** — 最具親和力及品牌延展性，適合較現代的 SaaS 形象。
4. **CurrentLedger** — 產品類別一目了然，但品牌個性較保守。
5. **LedgerShoal** — 水域風格鮮明，較適合強調鏈上資料編排與 Agent 協作。
